## [2.0.0] - 2026-01-16

### Initial Release

Professional security research tool for detecting CSRF vulnerabilities in web applications.

### Features
- Async web crawler with depth control
- Advanced CSRF vulnerability detection
- Multiple protection pattern recognition
- Dual-format reporting (Markdown + JSON)
- Severity classification system
- Interactive quickstart script
- Batch scanning capabilities
- Educational security lab demo
- Comprehensive documentation

### Detection Capabilities
- Token-based protection (CSRF tokens, nonces)
- SameSite cookie attributes
- Custom security headers
- Double-submit cookie patterns
- Origin/Referer validation

### Technical Stack
- Python 3.8+ with asyncio
- aiohttp for async HTTP
- BeautifulSoup4 for HTML parsing
- Type hints and dataclasses
- Abstract analyzer interface
